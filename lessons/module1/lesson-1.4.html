Before you can become a prompt engineering expert, you need to know your tools. This lesson walks you through the major AI platforms, how to access them, and how to set up your workspace for maximum productivity.

Major AI Platforms & LLMs

Different platforms offer different strengths. Understanding the landscape helps you choose the right tool for each task.

1. Google Gemini (Recommended for This Course)

Access: gemini.google.com (free tier available)

Best For: Research, coding, multimodal tasks (text + images), integration with Google Workspace

Key Features:

  • Large context window (handles lengthy documents)
  • Real-time web search grounding (reduces hallucination)
  • Can process images, videos, and audio
  • Code execution capabilities

2. ChatGPT (OpenAI)

Access: chat.openai.com (free tier with GPT-3.5, paid for GPT-4)

Best For: General writing, creative tasks, conversation, plugin ecosystem

Notable: Most widely adopted LLM; strong community and resources

3. Claude (Anthropic)

Access: claude.ai (free tier available)

Best For: Long-form content, detailed analysis, safety-focused tasks

Notable: Extremely large context window (200K+ tokens); excellent for processing entire books or codebases

4. Microsoft Copilot

Access: copilot.microsoft.com (integrated in Edge, Windows, Office)

Best For: Microsoft ecosystem integration, web search-enhanced responses

Setting Up Your Workspace

To maximize your prompt engineering efficiency, follow these setup steps:

Step 1: Create Accounts

  1. Sign up for Google Gemini at gemini.google.com (use your Google account)
  2. (Optional) Create accounts on ChatGPT and Claude for comparison
  3. Verify your email and explore the free tier limitations

Step 2: Understand the Interface

Key Interface Elements:

  • Prompt Input Box: Where you type your prompts (bottom of screen)
  • Response Area: Where the model's output appears
  • New Chat Button: Starts a fresh conversation (important: models remember context only within the same chat session)
  • Chat History: Sidebar showing previous conversations (for reference)
  • Settings/Model Selector: Choose between different model versions or adjust parameters

Step 3: Organize Your Workflow

Pro Tips for Organization:

  1. Use Descriptive Chat Titles: Rename your chats so you can find them later (e.g., "Marketing Copy - Product Launch" instead of "New Chat")
  2. Keep a Prompt Library: Save your best prompts in a separate document (Google Doc, Notion, or simple text file)
  3. Create Template Prompts: Build reusable templates for common tasks
  4. Use Folders/Tags: Some platforms allow organizing chats into categories

Understanding Chat Context & Memory

This is crucial to understand early:

Concept How It Works What It Means for You
Within-Session Memory Model remembers everything in the current chat You can reference earlier messages: "Make it shorter"
Between-Session Memory Completely reset when you start a new chat Must re-provide context in each new conversation
Context Window Limited token capacity (varies by model) Very long chats may "forget" early details

Free vs. Paid Tiers: What You Need to Know

Free Tier Limitations:

  • Rate limits (number of prompts per hour/day)
  • Access to older or smaller models
  • Slower response times during peak hours
  • No advanced features (plugins, file uploads, etc.)

Paid Tier Benefits ($20/month typical):

  • Access to most powerful models (GPT-4, Gemini Advanced)
  • Higher/unlimited usage limits
  • Priority access during high traffic
  • Advanced features (image generation, code execution, file analysis)
  • Better for professional/commercial use

For this course: The free tier is sufficient to learn and practice all concepts.

API Access (Advanced Option)

If you're a developer or want to integrate AI into applications:

  • Google AI Studio: ai.google.dev - Free API access with rate limits
  • OpenAI API: platform.openai.com - Pay-per-use pricing
  • Anthropic API: console.anthropic.com - Pay-per-use pricing

APIs allow you to programmatically send prompts and receive responses in your own applications.

Real-World Application: Your First Interaction

Practical Scenario:

You're a freelance graphic designer who needs to write a professional email to a potential client responding to their project inquiry.

Step-by-Step:

  1. Open Gemini (or your chosen platform)
  2. Click "New Chat" for a fresh conversation
  3. Use the prompt structure from Lesson 1.3
  4. Review the output and ask for revisions if needed
  5. Copy the final result for use

Mini Exercise: Platform Familiarization

Exercise Tasks:

  1. Create an account on Google Gemini (or your preferred platform)
  2. Start a new chat and send this simple prompt:
    "Explain the concept of prompt engineering in 3 simple sentences suitable for a complete beginner."
  3. Follow up with: "Now give me an example of a good prompt vs. a bad prompt."
  4. Start a new chat (fresh context) and try: "What did I just ask you?" Notice it has no memory of the previous chat.
  5. Rename your first chat to "Prompt Engineering Practice" so you can find it later.

Goal: Get comfortable navigating the interface and understanding conversation context.

Troubleshooting Common Issues

Issue Likely Cause Solution
"Too many requests" error Hit free tier rate limit Wait an hour or upgrade to paid tier
Model gives outdated info Training data cutoff date Use web search feature or verify externally
Response is cut off mid-sentence Hit token output limit Type "continue" or ask it to complete the response
Can't find old conversations Auto-deleted or not saved Always save important outputs externally

Key Takeaway:

Your tools are only as good as your understanding of them. Spend time exploring the interface, testing different features, and learning the limitations. This foundational knowledge will make you exponentially more effective as you progress through advanced techniques.
