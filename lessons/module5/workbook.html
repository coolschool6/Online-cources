Module 5 Workbook: Ethics, Monitoring, and Capstone

Operationalize AI responsibly. Use these exercises to design guardrails, evaluate quality, monitor performance, and ship a capstone project.

Lab 1: Harm & Bias Review

Identify potential harms and mitigations for your use case.

Template:
- Use case: ...
- Stakeholders: ...
- Risks: representational, allocative, privacy, security
- Mitigations: policy, technical, process
- Residual risk & escalation path

Lab 2: PII Redaction & Logging

Design input/output filters and safe logs.

PII list: email, phone, address, ssn
Policy: redact before storage, minimize retention to 7 days.
Exercise: demonstrate redaction patterns and safe logging JSON schema.

Lab 3: Prompt Injection Defense

Create a mutation set and detection/response policy.

Mutations: instruction override, URL bait, hidden controls, role confusion.
Policy: isolate untrusted content, quote-only mode, refuse escalation.
Test: 12 adversarial prompts and expected safe responses.

Lab 4: Quality Metrics & Dashboards

Define KPIs and sketch dashboards.

KPIs: helpfulness score, refusal rate, latency p95, error rate, complaint rate.
Dashboard JSON: panels[], metric, target, threshold, owner.
Alerting rules: who, when, how.

Lab 5: Evaluation Harness

Create a reference set and pass/fail checks.

Dataset JSONL: {"input":"...","ref":"...","policy":"..."}
Checks: correctness (0/1), policy compliance (0/1), structure (0/1).
Scoring prompt: compare MODEL vs REF and return a report object.

Lab 6: Rollout Strategy

Plan a staged rollout with guardrails.

Plan: canary (5%), feature flag, A/B, kill switch, rollback.
Checklist: preflight evals, observability, SLOs, on-call, comms plan.

Lab 7: Feedback Loop & Drift

Capture feedback and create a data flywheel.

Schema: {"prompt":"","output":"","thumb":"up|down","reason":"","category":""}
Process: triage weekly, curate dataset, retrain or reprompt.
Drift detection: moving average of helpfulness, alerts when -10%.

Capstone Project: Design, Evaluate, and Monitor an Assistant

Bring it all together with a scoped assistant.

Deliverables:
- PRD (goal, users, success metrics)
- Prompt(s) + schema(s)
- Safety policy + red team set
- Eval set (>=20), baseline results, and improvements
- Monitoring plan + dashboard sketch
- Post-launch checklist
