Modern LLMs like Google Gemini, GPT-4 Vision, and Claude can process images alongside text, unlocking powerful new capabilities. This lesson teaches you how to combine visual and textual inputs for tasks ranging from image analysis to document extraction, OCR, and visual question answering.

What is Multimodal Prompting?

Multimodal means the model can understand multiple types of input simultaneously‚Äîtext, images, and in some advanced cases, audio or video. This expands AI capabilities far beyond text-only interactions.

Current Multimodal Capabilities (2025):

  • Google Gemini: Text + images (photos, diagrams, PDFs, screenshots)
  • GPT-4 Vision (OpenAI): Text + images with high accuracy
  • Claude 3 (Anthropic): Text + images + multi-page PDF analysis
  • Emerging: Video understanding, audio analysis in some platforms

What You Can Do with Image Inputs

1. Image Description & Analysis

Use Cases:

  • Generate alt text for web accessibility (WCAG compliance)
  • Describe product photos for e-commerce listings
  • Identify objects, people, scenes, and actions in photos
  • Analyze image composition, quality, and aesthetics
  • Detect emotions, sentiment, or mood conveyed

Prompt example: "Describe this image in detail for someone who can't see it. Focus on: main subject, background elements, colors, mood, composition, and any text visible. Write it as descriptive alt text under 125 characters."

2. Text Extraction (OCR)

Extract text from:

  • Screenshots of documents, emails, code
  • Business cards (name, title, contact info)
  • Receipts and invoices (itemized expenses)
  • Handwritten notes (limited accuracy, best with neat handwriting)
  • Street signs, product labels, packaging
  • Whiteboards, presentation slides

Prompt example: "Extract all text from this receipt. Format as JSON with keys: storeName, date, items (array of {name, price}), subtotal, tax, total, paymentMethod."

3. Document Analysis

  • Summarize multi-page PDFs (reports, contracts, research papers)
  • Extract tables, charts, and graphs from scanned documents
  • Analyze data visualizations and explain trends
  • Compare two versions of a document side-by-side
  • Extract form data from filled-out PDFs

Prompt example: "This PDF contains our Q4 financial report. Extract data from the revenue chart on page 3 and create a summary table with: Quarter, Revenue ($M), Growth (%)."

4. Visual Question Answering

Ask specific questions about images:

  • "How many people are in this photo?"
  • "What brand logo appears on this product?"
  • "What time does the clock in this image show?"
  • "Is this image appropriate for children?"
  • "What's the error message in this screenshot?"
  • "What ingredients are visible in this recipe photo?"

5. Code from Screenshots

Upload a screenshot of a UI/design and request:

"Write the HTML and Tailwind CSS code to recreate this layout exactly. Include: responsive design for mobile, proper semantic HTML, accessibility attributes. Match colors, spacing, and typography as closely as possible."

Use case: Rapidly prototype designs, recreate layouts, convert mockups to code

Best Practices for Image Prompts

Be Specific About What You Want

Vague Prompt ‚ùå Specific Prompt ‚úÖ
"What's in this image?" "Identify all text visible in this store window display. List: brand names, product types, prices, promotional messages."
"Describe this product" "Describe this product for an e-commerce listing. Include: material, color(s), visible dimensions/size indicators, key features, and 3 suggested use cases."

Combine Multiple Images

Many models support multiple images in one prompt for comparison or sequence analysis:

  • Comparison: "Compare these two product photos. Which presents the product more effectively? Explain: lighting, composition, context, background choice."
  • Before/After: "These are before and after screenshots of a UI redesign. List all visual differences. Rate the improvement on: clarity, visual hierarchy, accessibility."
  • Sequence: "Analyze this sequence of 4 images showing a cooking process. Write step-by-step instructions matching each image."
  • Multi-angle: "These 3 photos show the same product from different angles. Write a comprehensive product description incorporating details from all views."

Real-World Applications

Application 1: Product Catalog Automation

Scenario: Bulk product listing creation

Challenge: Upload 50 product photos, need listing for each

Prompt (per image):

"Analyze this product image and generate JSON for an e-commerce listing:
{
'title': (descriptive 5-8 words),
'description': (2-3 sentences emphasizing benefits),
'suggestedCategory': ('Electronics' | 'Clothing' | 'Home' | 'Sports' | 'Beauty'),
'colors': (array of main colors visible),
'keywords': (array of 5-7 SEO keywords),
'suggestedPrice': (estimated price range based on perceived quality)
}

Be specific and accurate. Output ONLY JSON."

Result: Automated product data generation from photos alone

Application 2: Invoice/Receipt Processing

Scenario: Expense management automation

Upload: Photo of receipt (any quality, angle)

Prompt:

"Extract expense data from this receipt as JSON:

Application 3: UI/UX Feedback

Upload: Website screenshot or app interface

"Analyze this landing page design from a conversion optimization perspective:

6. Specific improvements: List 3 actionable changes with rationale

Limitations & Challenges

Be Aware of These Limitations:

  • Small/blurry text: Struggles with tiny fonts, low-resolution images, or poor image quality
  • Handwriting: Accuracy varies significantly; neat, large handwriting works better
  • Complex layouts: May misinterpret multi-column documents, overlapping text, or unusual formatting
  • Cultural context: May miss nuanced cultural references, idioms, or region-specific symbols
  • Image manipulation: Cannot reliably detect fake, edited, or deepfake images
  • Medical/legal: NOT certified for diagnostic decisions, legal judgments, or safety-critical applications
  • Faces/people: Some platforms restrict identification of specific individuals (privacy)

How to Upload Images

Platform-Specific Methods:

  • Google Gemini (web): Click attachment/paperclip icon, select image file
  • ChatGPT Plus: Click image icon (requires GPT-4 with Vision model selected)
  • Claude (Anthropic): Drag-and-drop image or click attachment button
  • APIs: Base64 encode image data or provide publicly accessible image URL

Supported formats (typically): JPG, PNG, WebP, GIF (static), PDF (some platforms), HEIC (iPhone photos, often)

Advanced: Chaining Image Analysis

Multi-step workflow example:

  1. Step 1: Upload document image ‚Üí "Extract all text from this document"

Key Takeaway:

Multimodal prompting bridges the gap between visual and textual information, unlocking workflows impossible with text alone‚Äîfrom automated data extraction to visual quality analysis and UI recreation. Always be explicit about what you want the model to "see" in the image, desired output format, and how to handle ambiguity. The more specific your instructions, the more reliable the extraction or analysis.
