Introduction: The Shift from Querying to Communication

Welcome to the foundation of Generative AI! For decades, human interaction with computers has been based on highly structured, rigid commands‚Äîthink programming languages, SQL queries, or button clicks. Generative AI marks a fundamental shift: we are moving from telling a machine exactly what to do via code, to communicating an intent via natural language.

This lesson defines what Generative AI is and establishes the core concept of Prompt Engineering as the new, vital skill for leveraging these tools.

1) Defining Generative AI

At its heart, Generative AI refers to deep-learning models capable of creating new content in response to a user's prompt. This content can be text, images, code, audio, or video.

The most popular form of Generative AI today is the Large Language Model (LLM), such as Gemini, which is trained on vast amounts of text data. An LLM doesn't store information like a database; rather, it learns the relationships and statistical patterns of human language.

How it Works: Next‚ÄëToken Prediction
  1. The model breaks your prompt (and its existing response) into tokens (words or sub‚Äëwords).
  2. It calculates the probability of which token should come next based on everything it has learned.
  3. It selects the next token, adds it to the response, and repeats the process, generating the response token by token.

The output is essentially the statistically most probable continuation of the input text, structured according to the patterns learned during training.

2) Discriminative vs. Generative AI: The Core Difference

Feature Discriminative AI Generative AI
Primary Goal Classification or Prediction Creation or Synthesis
Example Models Image Classifiers, Spam Filters, Regression LLMs, Image Generators (e.g., Stable Diffusion)
Output Type A label or score New, unique content
Question Answered ‚ÄúWhat is this? Is this A or B?‚Äù ‚ÄúWhat happens next? Create X based on Y.‚Äù

Key Takeaway: Discriminative AI separates and labels existing data. Generative AI creates data that did not exist before.
